# Mobile UI Checklist

Checklist для проверки мобильной версии интерфейса.

---

## Layout

- [ ] Интерфейс корректно масштабируется
- [ ] Нет горизонтального скролла
- [ ] Элементы не выходят за границы экрана

---

## Buttons and interactions

- [ ] Кнопки удобно нажимать пальцем
- [ ] Кнопки не перекрываются
- [ ] Кнопки имеют достаточный размер

---

## Forms

- [ ] Поля формы видны при открытии клавиатуры
- [ ] Кнопка Submit не перекрывается клавиатурой
- [ ] Поля удобно заполнять

---

## Performance

- [ ] Страницы быстро загружаются
- [ ] Анимации работают плавно
