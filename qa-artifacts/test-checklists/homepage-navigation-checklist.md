# Homepage Navigation Checklist

Checklist для проверки основной навигации сайта.

---

## Navigation menu

- [ ] Главное меню отображается на странице
- [ ] Все пункты меню кликабельны
- [ ] При клике происходит переход на соответствующую страницу
- [ ] Текущий раздел подсвечивается
- [ ] Меню корректно отображается на мобильных устройствах

---

## Dropdown menus

- [ ] Выпадающие меню открываются при клике
- [ ] Выпадающие меню закрываются при выборе пункта
- [ ] Выпадающее меню закрывается при клике вне области
- [ ] Пункты внутри dropdown кликабельны

---

## Links and navigation

- [ ] Логотип возвращает на главную страницу
- [ ] Footer содержит рабочие ссылки
- [ ] Нет битых ссылок (404)

---

## Responsiveness

- [ ] Меню адаптируется под мобильный экран
- [ ] Появляется мобильное меню (burger menu)
- [ ] Навигация удобна на небольших экранах
